    <h1>🌟 **Application de Gestion des Employés** 🌟</h1>

    <p><strong>Cette application web de gestion des employés</strong> est construite avec <strong>Jakarta EE v9</strong> et <strong>Maven</strong>. L'application permet aux utilisateurs d'effectuer des opérations CRUD (Créer, Lire, Mettre à jour, Supprimer) sur les enregistrements des employés.</p>

    <h2 class="section-title">🛠 **Fonctionnalités :**</h2>
    <ul>
        <li>✨ <strong>Ajouter un nouvel employé</strong></li>
        <li>🔍 <strong>Voir la liste des employés</strong></li>
        <li>✏️ <strong>Modifier les détails d'un employé existant</strong></li>
        <li>🗑️ <strong>Supprimer un employé</strong></li>
    </ul>

    <p>L'application utilise <strong>Jakarta EE</strong> pour la logique côté serveur et <strong>JSP</strong> pour l'interface utilisateur. <strong>Maven</strong> est utilisé pour la gestion des dépendances et pour la construction du projet.</p>

    <h2 class="section-title">🚀 **Technologies Utilisées :**</h2>
    <ul>
        <li><strong>Jakarta EE</strong> (Servlets, JSP)</li>
        <li><strong>Maven</strong> (Gestion des dépendances et construction)</li>
        <li><strong>Bootstrap</strong> (Stylisation de l'interface utilisateur)</li>
        <li><strong>MySQL</strong> (Base de données) – Vous pouvez utiliser <strong>XAMPP</strong> pour configurer facilement MySQL.</li>
        <li><strong>JDBC</strong> pour la connexion à la base de données</li>
    </ul>
